# CashFlow
Web app that makes managing expenses easier

### API endpoints: 
___
### Events

`GET` /api/events/

`GET` /api/events/event_id

`POST` /api/events/

`DELETE` /api/events/event_id
___
### Expenses
`GET` /api/expenses/

`GET` /api/expenses/event_id

`POST` /api/expenses/

`DELETE` /api/expenses/expense_id
___
### Payments
`GET`     /api/payments/event_id
